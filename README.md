# PocketPedia

A Vue 3 Pokedex application with PokeAPI.

Pokedex available in 3 languages:

- French
- English
- Japanese

## How to start

Install & run the project with:

```sh
npm install

npm run dev
```

And go to: [localhost:5173](http://localhost:5173/)

If you want to build the project:

```sh
npm run build
```
